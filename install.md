# Install

### Default Install

1. Download Release Version
2. Untar zip file
3. Run install

```bash
wget 
untar
./owl_install.sh
```

set license

```bash
./owlmanage.sh setlic=<lic>
```

### Docker Install

Reach out to OwlDQ team for docker file.  DockerHub coming soon.

| option | value |
